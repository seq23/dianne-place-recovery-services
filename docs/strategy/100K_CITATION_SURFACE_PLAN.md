# 100K Citation-Surface Plan

## Target

The target is 100K citation-surface opportunities in 6-12 months. This is a strategy target, not a guaranteed number of third-party LLM citations.

## What Counts

- Search impressions on citation-ready pages.
- LLM answer-surface appearances when observable.
- Earned mentions from public resource directories or partners.
- Indexed answer blocks and FAQ surfaces.
- Public citations to Dianne's Place pages.
- Internal citation opportunities produced by the source/claim/answer registry.

## Strategy

1. Build entity clarity around Dianne's Place Recovery Services, recovery housing, structured sober living, and referral support without forcing an unapproved local market.
2. Publish direct answer blocks that can be extracted by LLMs and search answer systems.
3. Build topical depth across women, minority communities, LGBTQ+ people, millennials, Gen Z, families, and referral partners.
4. Keep all clinical and licensure claims gated by the licensure mode switch.
5. Release one validated unit daily through 12/31/2026.
6. Use monthly pillar pages and quarterly white papers as higher-authority citation assets.
7. Refresh sources, schema, internal links, and answer blocks as release units rather than flooding thin pages.

## 9/10 AEO/SEO/GEO Requirements

- Query matrix exists.
- Entity registry exists.
- Source and claim registries exist.
- Answer block registry exists.
- Page family contracts exist.
- Route registry exists.
- Content atom registry exists.
- Public `answers.json`, `llms.txt`, and `llms-full.txt` exist.
- Content calendar is fully seeded from 2026-07-07 through 2026-12-31.
- Validators hard-check strategy, licensure, legal, content, admin, workflow, and AEO/SEO/GEO surfaces.

## Non-Negotiables

- No fabricated licensure.
- No fabricated outcomes.
- No unsupported clinical claims.
- No private or personal data as public signal.
- No spammy thin-page flooding.
